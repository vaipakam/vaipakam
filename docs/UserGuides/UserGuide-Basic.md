# User Guide — Basic mode (canonical source moved)

This file used to hold the canonical Basic-mode user guide. The text has
moved into the frontend bundle so it can be served localised by the
in-app /help/basic route.

**New location:** [`frontend/src/content/userguide/Basic.en.md`](../frontend/src/content/userguide/Basic.en.md)

To add a translation, drop a sibling file alongside the English source —
e.g. `Basic.es.md`, `Basic.ja.md` — and rebuild. The UserGuide page
picks up locale-matched files automatically and shows a
"translation pending" notice when only English is available.
